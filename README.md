This Unity project takes place in a futuristic space station where the player explores different rooms to collect glowing crystals. Each scene has unique elements such as dynamic objects, distinct lighting, wall fixtures, and machinery. The scenes also include characters that interact with objects in their surroundings. The project is meant to be an immersive and engaging experience for users.
